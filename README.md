# 🌑 Personal Portfolio Website

A modern, responsive, and visually clean **personal portfolio website** built using HTML, CSS, and JavaScript. This project showcases professional skills, services, education, and contact information in a structured and elegant layout.

Designed with a strong dark theme and smooth user experience, this portfolio serves as a digital identity for developers who want to present their work effectively.

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/tharuusamaraweera/portfolio_website.git
```

### 2. Open the project

* Open the folder in **VS Code** or any editor
* Run `index.html` in your browser

### 3. Optional (Recommended)

Use **Live Server** extension in VS Code for better development experience.

---

## ✏️ Customization Guide

You can easily personalize this portfolio:

### Change Name & Content

Edit inside `index.html`

```html
<h1>Hi, It's <span>Your Name</span></h1>
```

### Replace Profile Image

```html
<img src="your-image.jpg">
```

### Update Social Links

```html
<a href="https://github.com/yourusername">
```

### Modify Styles

Edit `style.css` to:

* Change colors
* Adjust layout
* Add animations
